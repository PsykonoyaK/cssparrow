Advantages

Light

CsSparrow is one of the lightest css framework. Because each line count, we optimize all the code.

Flexible

To feet your needs we use the power of Sass to change the look of your web project easily.

Responsive

Because each devices deserve the best user experience, CsSparrow implement responsive features and fluid grid.

Main features

Grid

This responsive grid works with overflow hidden to reset float elements and use css box-sizing to manage columns.

IE complient

Cssparrow also works with Internet Explorer until 7.

Widgets

Take advantage of Responsive navigation, fluid Thumbnail and buttons library.

theme tools

Change colors, margins and fonts easily with variable.scss.

How it works ?

Powered by Sass, Cssparrow give you the most optimized, flexible and responsive Cascading Style Sheets.

You just need to install Sass and create and implement your own sass documents in global.scss. Then you will be able to take the control with variable.scss and use @extend sass feature for a better maintainability.

You also can use Cssparrow without Sass like a traditionnal css framework.

CsSparrow can be adapted to all of your web project. Use the power of Sass to change the look of your web site easily.
