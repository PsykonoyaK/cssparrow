CsSparrow

Cssparrow est un framework css responsive construit avec SASS et fonctionnant avec SASS. La structure de Cssparrow à été pensé pour évoluer et s'adapter à vos besoins tout en garantissant une bonne maintenabilité. Tirez partie des variables SASS et appropriez vous celles de cssparrow en les intégrants à vos feuilles de styles scss.

Advantages

Maintainable

Evolved according to your needs and stay easy to used and maintain.

Flexible

To feet your needs we use the power of Sass to change the look of your web project easily.

Responsive

Because each devices deserve the best user experience, CsSparrow implement responsive features and fluid grid.

Main features

Grid

This responsive grid works with overflow hidden to reset float elements and use css box-sizing to manage columns.

IE complient

Cssparrow also works with Internet Explorer until 7.

theme tools

Change colors, margins and fonts easily with variable.scss.
